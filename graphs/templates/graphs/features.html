GraphSpace Features

Interact with graphs

  • View, save, and customize layouts of graphs
  • Sequentially step through subgraphs of the entire graph

Share graphs

  • Create groups and add collaborators
  • Share graphs between all members of a group
  • Share layouts with collaborators
  • Share graphs with the world

Search graphs

  • Search for elements inside of graphs
  • Search for graphs that contain elements

Organize graphs

  • Tags allow for organizing multiple graphs
  • Quickly search for all graphs that match tag

Programmatically communicate with GraphSpace

  • Use the comphrehensive REST API
  • Customize graphs and faciliate graph uploads
  • Create scripts to interact with features of GraphSpace
